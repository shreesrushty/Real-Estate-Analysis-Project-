# Real Estate Investment Analysis Dashboard


 # Project Overview

This project is a Real Estate Data Analysis and Investment Dashboard built using Python and Streamlit. It focuses on analyzing property data to extract meaningful insights such as pricing trends, location impact, property features, and investment potential.

The project helps users understand how various factors like city, size, amenities, and transport accessibility affect property prices.

# Objectives
Clean and preprocess real estate dataset
Perform Exploratory Data Analysis (EDA)
Identify key factors affecting property prices
Visualize trends using charts and graphs
Build an interactive dashboard using Streamlit
Enable city-wise filtering and analysis
